# Mental Health Support System

Fine-tuned DistilBERT classifier + FAISS semantic retrieval + Streamlit UI.  
Classifies user statements into 7 mental health categories and surfaces curated coping resources.

---

## Architecture

```
user statement
      │
      ▼
┌─────────────┐        ┌──────────────────────────┐
│ DistilBERT  │        │ DistilBERT (encoder only) │
│ classifier  │        │ → CLS embedding (768d)    │
└──────┬──────┘        └───────────┬──────────────┘
       │                           │
  predicted label             FAISS flat-L2 search
  + probabilities              over dataset embeddings
       │                           │
       ▼                           ▼
 resource lookup           top-k similar statements
 (JSON mapping)
       │
       └─────────────────────────────────────────┐
                                                 ▼
                                         PipelineResult
                                         (Streamlit UI)
```

## Classes

| Label | Description |
|---|---|
| Anxiety | Generalised anxiety, panic, worry |
| Depression | Depressive episodes, low mood |
| Stress | Situational stress, overwhelm |
| Bipolar | Manic/depressive cycling |
| Suicidal | Suicidal ideation or intent |
| Personality disorder | BPD, NPD, etc. |
| Normal | No clinical signal detected |

## Setup

```bash
# 1. Clone / download the repo
git clone <url>
cd mental-health-semantic-search

# 2. Install dependencies (Python 3.10+)
pip install -r requirements.txt

# 3. Place dataset in project root
cp /path/to/Combined_Data.csv .

# 4. Run all pipeline stages
python run_pipeline.py --data Combined_Data.csv

# 5. Launch UI
streamlit run app.py
```

**GPU recommended** for fine-tuning (stages 2-3). CPU works but is slow.  
For free GPU: open `notebooks/` in Google Colab.

## Running Individual Stages

```bash
# Stage 1 only
python -m src.data_prep Combined_Data.csv

# Stage 2 only
python -m src.finetune --train data/train.csv --val data/val.csv

# Stage 3 only
python -m src.build_index --dataset data/dataset.csv

# Skip training (use existing model), rebuild index
python run_pipeline.py --data Combined_Data.csv --skip-train
```
